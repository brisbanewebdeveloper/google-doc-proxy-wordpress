This plugin provides to manage to display Google Docs via Gogle Doc Proxy.

== Description ==

[Google Doc Proxy](http://google-doc-proxy.hironozu.com/) is a proxy that delivers your Google Doc to anywhere with appropriate HTML tags as JSON.
This plugin provides you to manage to display Google Docs via Gogle Doc Proxy.

Google Doc Proxy is useful for the following work environments:

*   Person/Organisation who is responsible for development work is different from the one doing content entry work.
*   You are web developer and would not like your client to see Admin area until the site is launched.

More details can be found at [Google Doc Proxy](http://google-doc-proxy.hironozu.com/).

== Installation ==

1. Upload `google-doc-proxy.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php do_action('gdoc_prox_display', 'document_id'); ?>` in your templates
